========================================
  Educational Summary
========================================

Key Concepts Demonstrated:

1. Regression vs Classification:
   - Regression: Predict continuous values (sin(x) in [-1, 1])
   - Classification: Predict discrete classes (like MNIST digits)
   - Same network architecture, different interpretation!

2. Universal Function Approximation:
   Neural networks can approximate ANY continuous function!
   This is a fundamental theorem in neural network theory.
   We demonstrated this by learning sin(x).

3. Interpolation vs Extrapolation:
   - Interpolation MAE: 0.0346
   - Extrapolation MAE: 0.2217
   Neural networks struggle with extrapolation!
   They work best within the training data range.

4. Activation Function Choice:
   We used tanh for all layers because:
   - Output range [-1, 1] matches sin(x) range
   - Zero-centered (better for learning)

5. Practical Applications:
   Function approximation is useful for:
   - Time series prediction (stock prices, weather)
   - Physics simulations (approximate complex equations)
   - Control systems (learn system dynamics)
