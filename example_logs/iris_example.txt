========================================
  Educational Summary
========================================

Key Concepts Demonstrated:

1. Tabular Data Classification:
   Neural networks work on any type of data, not just images.
   Iris has 4 numerical features (flower measurements).

2. Feature Normalization:
   We normalized features to [0, 1] range before training.
   This is crucial for neural networks to train effectively.
   Without normalization, features with larger values dominate.

3. Small Dataset Challenges:
   Iris has only 150 samples (120 train, 30 test).
   Small datasets are prone to overfitting.
   We use a simple architecture (6 hidden neurons) to avoid this.
   Fewer neurons = less capacity to memorize = better generalization.

4. Multi-Class Classification:
   3 output neurons, one per class (one-hot encoding).
   Predicted class = neuron with highest activation.

5. Classical ML Dataset:
   Iris is a benchmark dataset from 1936!
   It's often used to compare different ML algorithms.
   Neural networks are actually overkill for Iris,
   but it's great for learning and demonstration.
